## AWS Services Overview

Here are some of the key AWS services that I am learning about:

### 1. Compute Services
- **Amazon EC2**: Scalable virtual servers in the cloud.
- **AWS Lambda**: Run code without provisioning or managing servers.

### 2. Storage Services
- **Amazon S3**: Scalable object storage for data backup and archiving.
- **Amazon EBS**: Block storage for use with Amazon EC2.

### 3. Database Services
- **Amazon RDS**: Managed relational database service.
- **Amazon DynamoDB**: NoSQL database service for high-performance applications.

### 4. Networking Services
- **Amazon VPC**: Isolated cloud resources for your applications.
- **AWS CloudFront**: Content delivery network (CDN) for faster content delivery.

### 5. Security Services
- **AWS IAM**: Manage access to AWS services and resources securely.
- **AWS Shield**: DDoS protection for applications.
